Sudoku Evolution

Política de Privacidade

Introdução

Seu jogo, sua privacidade

Esta Política de Privacidade descreve como o aplicativo Sudoku Evolution, desenvolvido por Nave8, coleta, usa e protege as informações dos usuários. Ao utilizar o app, você concorda com os termos descritos neste documento.

🔒 Resumo simples: Não coletamos nome, e-mail, telefone ou qualquer dado pessoal identificável. Seu progresso é salvo de forma anônima e segura.

01

Dados Coletados

O Sudoku Evolution coleta apenas os dados estritamente necessários para o funcionamento do aplicativo:

  • Identificador anônimo de usuário — gerado automaticamente pelo Firebase Authentication, sem qualquer informação pessoal associada
  • Progresso de jogo — nível, XP, streaks, histórico de partidas e configurações salvas no Firestore
  • Dados de uso anônimos — eventos de navegação e interação coletados pelo Firebase Analytics para fins de melhoria do app
  • Versão do app e plataforma — utilizados para controle de atualizações via Firebase Remote Config
02

Como Usamos os Dados

Os dados coletados são utilizados exclusivamente para:

  • Salvar e sincronizar seu progresso de jogo entre sessões
  • Calcular streaks, XP e histórico do Desafio Diário
  • Exibir notificações de lembrete (configuráveis pelo usuário)
  • Garantir que você está usando uma versão atualizada e segura do app
  • Analisar de forma agregada e anônima como o app é utilizado, para melhorar a experiência
03

Autenticação Anônima

O Sudoku Evolution utiliza o Firebase Authentication com autenticação anônima. Isso significa que você não precisa criar uma conta, fornecer e-mail ou senha.

Um identificador único e aleatório é gerado automaticamente para armazenar seu progresso. Este identificador não contém nenhum dado pessoal e não pode ser utilizado para identificar você como pessoa.

⚠️ Atenção: Como a autenticação é anônima, desinstalar o aplicativo apaga o vínculo com seu progresso. Atualizações do app preservam seus dados normalmente.

04

Compartilhamento de Dados

Não vendemos, alugamos ou compartilhamos seus dados com terceiros para fins comerciais ou publicitários.

Os dados são processados pelos seguintes serviços do Google Firebase, sujeitos à Política de Privacidade do Google :

  • Firebase Authentication — identificação anônima
  • Cloud Firestore — armazenamento do progresso
  • Firebase Analytics — análise de uso anônima
  • Firebase Remote Config — controle de versões
05

Notificações

O aplicativo pode enviar notificações locais para lembrá-lo de jogar o Desafio Diário e manter seu streak ativo. Estas notificações são geradas no próprio dispositivo e não envolvem servidores externos.

Você pode desativar as notificações a qualquer momento nas configurações do seu dispositivo Android.

06

Retenção e Exclusão de Dados

Seus dados de progresso são armazenados enquanto você utilizar o aplicativo. Para solicitar a exclusão completa dos seus dados, entre em contato conosco pelo e-mail abaixo e processaremos sua solicitação em até 30 dias.

07

Crianças e Menores de Idade

O Sudoku Evolution não é direcionado a crianças menores de 13 anos e não coleta intencionalmente dados de menores. Se você acredita que um menor forneceu dados ao app, entre em contato conosco para que possamos removê-los.

08

Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. Quando houver mudanças significativas, atualizaremos a data de revisão abaixo e, se necessário, notificaremos os usuários através do aplicativo.

Recomendamos que você revise esta página ocasionalmente para se manter informado.

09
